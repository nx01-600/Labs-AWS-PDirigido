PLAN DE EJECUCIÓN DEL PROYECTO
Laboratorios Virtuales de Redes en AWS
===============================================

ENFOQUE DEL PROYECTO:
---------------------
- **8 Laboratorios** completos y detallados
- **100% GRATUITO** (solo Free Tier de AWS)
- **SIN CAPTURAS** (todo explicado textualmente)
- **Listo para entregar** (sin trabajo adicional)

ESTRUCTURA DEL PROYECTO:
------------------------

1. LABORATORIOS (Archivos LaTeX individuales)
   - Cada laboratorio será un documento PDF profesional
   - Teoría extensa, pasos MUY detallados (sin capturas)
   - Diagramas conceptuales con TikZ o ASCII art
   
2. CUESTIONARIOS Y EVALUACIONES
   - Archivo compilado con 120+ preguntas tipo Kahoot/Quiz
   - Integradas en cada laboratorio
   
3. DOCUMENTACIÓN FINAL
   - Memoria técnica del proyecto
   - Manual de implementación para profesores
   - README completo


LABORATORIOS PROPUESTOS (8 Labs - 100% GRATUITOS):
--------------------------------------------------

LAB 1: Introducción a AWS y Configuración de Cuenta
- Guía completa de creación de cuenta AWS (paso a paso textual)
- Configuración de IAM (usuarios, grupos, políticas) - Free Tier
- Configuración de MFA y seguridad básica
- Navegación por la consola AWS
- **100% Gratuito - No requiere recursos**

LAB 2: Fundamentos de Amazon VPC
- Conceptos teóricos: VPC, subnets, CIDR
- Guía detallada de creación de VPC personalizada
- Configurar subnets públicas y privadas (explicación paso a paso)
- Tablas de ruteo - Teoría y configuración
- **100% Gratuito - VPC no tiene costo**

LAB 3: Conectividad a Internet en VPC
- Internet Gateway (configuración detallada)
- NAT Gateway vs NAT Instance (solo teoría - NAT Gateway NO es gratuito)
- Elastic IP (explicación, evitar uso prolongado)
- Configuración de rutas para acceso a internet
- **Gratuito usando solo Internet Gateway**

LAB 4: Amazon EC2 y Seguridad de Red
- Lanzar instancias EC2 t2.micro/t3.micro (Free Tier)
- Security Groups y reglas (explicación detallada)
- Network ACLs (configuración y diferencias con SG)
- **Gratuito con Free Tier (750 horas/mes)**

LAB 5: Seguridad Avanzada en Redes AWS
- Security Groups avanzados (múltiples capas)
- VPC Flow Logs (teoría y configuración)
- Análisis de logs con CloudWatch Logs Insights
- Mejores prácticas de seguridad
- **Gratuito con límites de Free Tier**

LAB 6: VPC Peering (Conceptual)
- Conceptos de VPC Peering
- Casos de uso y limitaciones
- Diseño de arquitectura con peering
- Configuración teórica paso a paso
- **Gratuito - Sin transferencia de datos**

LAB 7: Monitoreo de Redes con CloudWatch
- CloudWatch para métricas de red
- Análisis de VPC Flow Logs
- Creación de alarmas
- Dashboard de monitoreo
- **Gratuito con Free Tier (10 métricas, 10 alarmas)**

LAB 8: Proyecto Final - Arquitectura Completa
- Diseñar arquitectura de red completa en AWS
- Integrar todos los conceptos (VPC, EC2, Seguridad)
- Documentar decisiones de diseño
- Análisis de costos y optimización
- **Proyecto teórico con recursos gratuitos**


CUESTIONARIOS (Por cada laboratorio):
--------------------------------------

Cada lab tendrá:
- 10-15 preguntas de opción múltiple
- 3-5 preguntas de verdadero/falso
- 2-3 preguntas de escenarios prácticos
- 1-2 preguntas de troubleshooting

Tipos de preguntas:
- Conceptuales (teoría de redes y AWS)
- Prácticas (configuración específica)
- Troubleshooting (identificar problemas)
- Diseño (mejores prácticas)

9. Preguntas de Reflexión y Cuestionario
   - 10-15 preguntas integradas en el lab
   - Ejercicios conceptuales

10. Referencias y Recursos
