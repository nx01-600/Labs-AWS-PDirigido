LABORATORIOS FINALES (8 Labs):
-------------------------------

LAB 1: Introducción a AWS y Creación de Cuenta
DURACIÓN: 60-90 minutos | COSTO: $0.00
Contenido:
- Guía COMPLETA de creación de cuenta AWS (cada paso explicado)
- Navegación por la consola (menús, servicios, regiones)
- Configuración de IAM: usuarios, grupos, políticas
- Configuración de MFA para seguridad
- Configuración de alertas de facturación
- Comprensión del Free Tier
NOTA: Todo explicado paso a paso sin capturas

LAB 2: Fundamentos de Amazon VPC
DURACIÓN: 90-120 minutos | COSTO: $0.00
Contenido:
- Teoría completa: ¿Qué es una VPC?, CIDR, Subnetting
- Creación de VPC personalizada (10.0.0.0/16)
- Creación de subnets públicas y privadas
- Configuración de tablas de ruteo
- Entender el router virtual de AWS
NOTA: VPC no tiene costo

LAB 3: Conectividad a Internet en VPC
DURACIÓN: 90 minutos | COSTO: $0.00
Contenido:
- Configuración de Internet Gateway (IGW)
- Adjuntar IGW a VPC
- Configurar rutas para acceso a internet
- Teoría de NAT Gateway (NO IMPLEMENTAR - cuesta dinero)
- Diferencias entre NAT Gateway e Internet Gateway
- Elastic IP (teoría, no usar en producción)
NOTA: Solo usar IGW (gratuito), NAT Gateway solo teoría

LAB 4: Amazon EC2 y Seguridad de Red
DURACIÓN: 120 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- Lanzar instancia EC2 t2.micro o t3.micro (Free Tier)
- Configuración de Security Groups (reglas inbound/outbound)
- Network ACLs: diferencias con Security Groups
- Stateful vs Stateless
- Mejores prácticas de seguridad
- Conectarse a instancia vía SSH
NOTA: 750 horas/mes gratis con Free Tier

LAB 5: Seguridad Avanzada en Redes
DURACIÓN: 90 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- Security Groups multicapa
- VPC Flow Logs (configuración y análisis)
- CloudWatch Logs Insights (consultas básicas)
- Detectar tráfico anómalo
- Principio de mínimo privilegio
- Defense in depth
NOTA: Flow Logs gratuito con límites de Free Tier

LAB 6: VPC Peering (Teórico-Práctico)
DURACIÓN: 90 minutos | COSTO: $0.00
Contenido:
- Conceptos de VPC Peering
- Casos de uso reales
- Limitaciones (no transitivo)
- Diseño de arquitectura con peering
- Configuración paso a paso (sin transferir datos)
- Alternativas: Transit Gateway (solo teoría)
NOTA: Crear peering es gratis, transferir datos cuesta

LAB 7: Monitoreo de Redes con CloudWatch
DURACIÓN: 90 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- CloudWatch para métricas de red
- Análisis de VPC Flow Logs
- Creación de alarmas (máx 10 gratis)
- Dashboard de monitoreo
- CloudWatch Logs Insights
- Mejores prácticas de monitoreo
NOTA: 10 alarmas y 10 métricas gratis

LAB 8: Proyecto Integrador - Arquitectura Completa
DURACIÓN: 180 minutos | COSTO: $0.00
Contenido:
- Diseñar red corporativa completa en AWS
- Integrar: VPC, subnets, IGW, EC2, Security Groups
- Documentar decisiones arquitectónicas
- Análisis de costos y optimización
- Escalabilidad y alta disponibilidad (teórico)
- Presentar solución profesional
NOTA: Proyecto de diseño con implementación básica
