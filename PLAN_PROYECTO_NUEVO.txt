PROYECTO DIRIGIDO - LABORATORIOS DE REDES EN AWS
===============================================
Universidad Javeriana Cali
Redes de Nueva Generación
Director: Carlos Olarte
Diciembre 2025

DESCRIPCIÓN DEL PROYECTO:
------------------------
Desarrollo de 8 laboratorios prácticos en AWS diseñados para fortalecer competencias en redes mediante servicios de nube, utilizando exclusivamente AWS Free Tier.

AUTORES:
--------
- Nicolás Carreño Tascón
- Juan Manuel Canchala Jiménez

LABORATORIOS IMPLEMENTADOS (8 Labs):
------------------------------------

LAB 1: Introducción a AWS y Creación de Cuenta
DURACIÓN: 60-90 minutos | COSTO: $0.00
Contenido:
- Creación y configuración de cuenta AWS
- Navegación por la consola AWS
- Configuración de IAM: usuarios, grupos, políticas
- Configuración de MFA para seguridad
- Configuración de alertas de facturación
- Comprensión del Free Tier

LAB 2: Fundamentos de Amazon VPC
DURACIÓN: 90-120 minutos | COSTO: $0.00
Contenido:
- Teoría de VPC, CIDR, Subnetting
- Creación de VPC personalizada (10.0.0.0/16)
- Creación de subnets públicas y privadas en multi-AZ
- Configuración de tablas de ruteo
- Arquitectura de red escalable

LAB 3: Conectividad a Internet en VPC
DURACIÓN: 90 minutos | COSTO: $0.00
Contenido:
- Configuración de Internet Gateway (IGW)
- Configuración de rutas para acceso a internet
- Teoría de NAT Gateway
- Diferencias entre NAT Gateway e Internet Gateway
- Mejores prácticas de conectividad

LAB 4: Amazon EC2 y Seguridad de Red
DURACIÓN: 120 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- Lanzamiento de instancias EC2 t2.micro
- Configuración de Security Groups
- Network ACLs y diferencias con Security Groups
- Implementación de arquitectura bastion host
- Conexión SSH segura

LAB 5: Seguridad Avanzada en Redes
DURACIÓN: 90 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- Security Groups multicapa
- VPC Flow Logs
- CloudWatch Logs Insights
- Análisis de tráfico de red
- Principio de mínimo privilegio

LAB 6: VPC Peering
DURACIÓN: 90 minutos | COSTO: $0.00
Contenido:
- Conceptos de VPC Peering
- Configuración de peering entre VPCs
- Limitaciones y consideraciones
- Casos de uso empresariales
- Alternativas de conectividad

LAB 7: Monitoreo de Redes con CloudWatch
DURACIÓN: 90 minutos | COSTO: $0.00 (Free Tier)
Contenido:
- CloudWatch para métricas de red
- Análisis de VPC Flow Logs
- Creación de alarmas
- Dashboards de monitoreo
- Mejores prácticas de observabilidad

LAB 8: Proyecto Integrador - Arquitectura Completa
DURACIÓN: 180 minutos | COSTO: $0.00
Contenido:
- Arquitectura 3-tier completa en AWS
- Integración de VPC, subnets, IGW, EC2, Security Groups
- Implementación de seguridad multicapa
- Monitoreo completo con CloudWatch
- Documentación arquitectónica

CUESTIONARIOS:
--------------

- 15 preguntas por laboratorio
- 10 opción múltiple (A, B, C, D)
- 3 verdadero/falso
- 2 escenarios/troubleshooting

Cada pregunta incluye:
- Enunciado claro
- Opciones de respuesta
- Respuesta correcta marcada
- Explicación detallada
